Coding Tutorials!

The first thing you need to know is how to establish variables
Variables are a way the computer stores information. Which can be stored in many different ways, but we will only focus on a few:
Integers, which are basic number without decimals, and Strings, which are words and letters
To define a variable, you need to use the code 'Dim' as such
'Dim N as Integer'
With 'N' being the name of the object the computer will remember


The next thing to know if how to make decisions.
To make decisions, you will need either an If/Then statement, or a Select Case Statement.
An If/Then Statement is used to describe how If a specific option is inputted, then the computer will provide a specific output, as such
If N = 15 Then N = 15 end If
Another way to make decisions is with a Select Case Statement, which defines multiple different inputs creating multiple different outputs
The way that a Select Case is different is that an If/Then Statement has limited options to specific inputs, but a Select Case Statement can have as many inputs and outputs as you can type! As such:
Select Case Case N = 15 N = 15 Case N = 17 N = 17 ... Case N = 100 N = 100 end Select


Another interesting coding statement are Loops

The last important piece of information needed are inputs and outputs
Inputs are what the user of your program decide based on the options that you provide, and can be read as such
N = Console.ReadLine()
This will make the computer read what the user input and define the variable as that input
An output is what the computer tells the user, either based on what the user chose as an input, or programmed whenever the programmer decides. As such:
Console.WriteLine("This is the Output")
This will read out to the user 'This is the Output'
